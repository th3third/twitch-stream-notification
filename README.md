# Twitch Stream Notification PhoneGap Template

A PhoneGap Twitch Stream Notification template using OneSignal for push notifications.

## Usage

Build as you would a normal PhoneGap application. You will need to enter the following pieces of information:

* index.js: OneSignal client information.
* index.js: stream name
* config.xml: app id and name
* index.html: app name
